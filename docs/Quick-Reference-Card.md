# ACG Lead Package - Quick Reference Card

## Daily Workflow

```
1. CHECK LEADS    →  npm run summary
2. PICK A-LEAD    →  Start with highest violation count
3. RESEARCH       →  Find actual contact name/email
4. OPEN FOLDER    →  open output/<folder-name>
5. CUSTOMIZE      →  Update email-draft.txt with real name
6. SEND EMAIL     →  Attach PDF, send from team@
7. LOG            →  Update Lead Tracker spreadsheet
8. REPEAT         →  Move to next A-lead
```

---

## Commands Cheat Sheet

| What You Want | Command |
|---------------|---------|
| Scan a new site | `npm run lead-package -- <URL> "Name" "email"` |
| View all leads | `npm run summary` |
| Open output folder | `open output/` |

## Lead Grades

| Grade | What It Means | Action |
|-------|---------------|--------|
| **A** | 10+ critical/serious | Contact TODAY |
| **B** | 5-9 critical/serious | Contact within 48 hrs |
| **C** | Under 5 critical/serious | Batch outreach |
| **Skip** | 0 violations | Don't contact |
